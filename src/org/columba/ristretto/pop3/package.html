Low level implementation of the Post Office Protocol Version 3 (POP3) as specified by RFC 1939. The main class in this package is the POP3Protocol. All methods are strongly related to the commands defined in the RFC. If you have any problems using the protocol consult the accorind RFCs. Some more important RFCs:
  • RFC 1939 - Post Office Protocol Version 3
  • RFC 2249 - POP3 Extension Mechanism
  • RFC 2595 - Using TLS with IMAP, POP3 and ACAP
  • RFC 1734 - POP3 AUTHentication command
